# Cambio Automático

Este proyecto busca desarrollar un sistema basado en el microcontrolador Espressif ESP32 con el cual gestionar el cambio de marchas de un coche de Formula Student.

## Cosas Importantes

Muy probablemente se mantendrá la extensión **.ino** por lo que la opción *de facto* para editar el programa será **AruinoIDE**, aunque se pueden utilizar otros editores u entornos de desarrollo.
